Create a private GitHub repository

This is where your knowledge will live. It's private — only you can access it.

1 Go to GitHub and create a new repository Open GitHub ↗
2 Name it anything — e.g. my-brain
3 Make sure Private is selected
4 Click "Create repository" — leave everything else as-is
5 Copy the URL from your browser's address bar and paste it below

Create a Personal Access Token

This gives Second Brain permission to read and write your private repository on GitHub.

1 Open the GitHub token page Open Token Page ↗
2 Give it any name — e.g. second-brain
3 Under "Select scopes", make sure repo is checked ☑
4 Scroll down and click "Generate token"
5 Copy the token shown — it starts with ghp_
🔒 Stored locally on this computer only. Never transmitted anywhere except GitHub.
